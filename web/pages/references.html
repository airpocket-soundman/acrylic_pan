資料と解析の再現方法

ボード、センサ、Solist AIの仕様根拠と、振動モード解析を再実行する手順をまとめています。

Dockerで解析する

Docker DesktopのLinuxエンジンを起動し、プロジェクトルートで実行します。

docker build -t acrylic-pan-sim .
docker run --rm -v "${PWD}/web/assets/simulation:/workspace/web/assets/simulation" acrylic-pan-sim

PowerShell用の記述です。解析結果のJSONとmode-1.svg～mode-8.svgが更新されます。

解析ファイル

物理モデルsim/modal_plate.py薄板固有値、中央1センサのモード結合強度、8打撃点のRMS振動強度図
AI予備評価sim/solist_feasibility.py1隠れ層ELMの合成特徴による成立性確認
数値結果web/assets/simulation/results.json材料条件、固有振動数、8打撃点の強度
