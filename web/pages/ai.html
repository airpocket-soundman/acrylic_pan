05 / Solist AI model

隠れ層は1層で大丈夫か。

結論は、8点識別には十分成立する構成です。Solist AIは3層FFNN、つまり「入力層＋隠れ層1層＋出力層」のELM系モデルです。深さではなく、振動波形から位置差が見える特徴を作れるか、隠れノード数と乱数Seedが十分かが性能を決めます。

YES

まず隠れ32～64ノードで評価できます。
この用途は画像認識のような階層的特徴抽出ではなく、短時間波形／周波数帯強度から8つの既知領域を分離する小規模問題です。

Solist AIの学習構造

INPUT

振動特徴 x

時間波形128点、または選別したモード／FFT帯域。

→ α fixed
HIDDEN × 1

非線形写像 h

乱数固定重み。Hard sigmoid等。Seedで分離性能が変化。

→ β learned
OUTPUT

8空間スコア

隠れ→出力の重みだけを教師あり学習。

y = h(x · α) · β　　α：乱数固定　β：教師データから更新

推奨する2段階モデル

モデルA128入力 + 64隠れ + 8出力 = 200符号付きZ時間波形。位相と立上り差を保持
モデルB64入力 + 64隠れ + 8出力 = 136周波数帯／モード強度を64特徴以内へ圧縮
上限目安入力 + 隠れ + 出力 ≤ 570最終値はSim表示のAI RAM使用量で確認
ノード数には十分な余裕128波形入力案でも合計200で、資料の570目安の35%。32隠れなら168です。1層しかないことより、入力の正規化と打撃強度・タイミングずれへの耐性が重要です。

予備シミュレーション

100 × 20 mm面固定の物理解析から得た18モードの加速度結合強度を入力し、強度変動とノイズを加えた合成データで、Solist AI互換の1隠れ層・Hard Sigmoid・固定ランダム入力重みを評価しました。

100%SINGLE · HIDDEN 64

単打8点、10 Seed。隠れ8では平均91.6%のため非推奨。

88.8%1 + 2 HITS · HIDDEN 64

単打8＋同時2点28組の完全一致率。ラベルF1は96.3%。

90TOTAL NODES

入力18＋隠れ64＋出力8。570目安内。

これは1隠れ層の表現力が理想化した線形重畳特徴を分離できることだけを示す予備結果です。同時2点は単打より難しく、実物の減衰、打撃時差、固定ばらつき、打撃位置ずれを加えると低下する可能性があります。

出力は座標2値より8スコアを先に採用

8出力：推奨

単打はone-hot、同時2点は該当する2出力を1にしたmulti-hotで学習します。最大2つの有効スコアが音階、全出力がヒートマップになります。

pᵢ = normalize(scoreᵢ)
x̂ = Σ pᵢ xᵢ,　ŷ = Σ pᵢ yᵢ

2出力：不向き

`x_norm, y_norm`の直接回帰では、同時2点が中間の1点へ潰れます。複数打撃を扱うには8出力が必要です。

同時2点と0.1秒連打

同時2点：multi-label

8点から2点を選ぶ組合せは28通り。しきい値を超えた上位2出力を、ポリフォニック音源で独立に同時発音します。

target = [0, 1, 0, 0, 0, 1, 0, 0]

100 ms以上：別イベント

20 msの解析窓後、打撃開始から40 ms以内に再アーム。前打の残響を含む連打データで2打目を学習し、前の音を止めずに重ねます。

1層モデルで失敗しやすい条件

  • FFT振幅だけで左右鏡像がほぼ同じになる
  • 打撃の強弱が位置差より大きく、正規化していない
  • 学習と評価を同じ収録セッションからランダム分割する
  • 単一Seedだけで評価し、乱数固定重みへの偶然の適合を見落とす
  • 中央センサが節になる周波数を主特徴として選ぶ
