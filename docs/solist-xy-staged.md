# Solist-AI 多段XY座標回帰（実験保留）

## 結論

64ユニット幅を7層重ねた `128-64-64-64-64-64-64-64-2` 座標回帰モデルは、
Bfloat16エミュレーション上ではLOSO平均距離 **7.10 mm**、90%点 **16.46 mm**、
25 mm以内 **94.1%** となり、PC側の広幅モデルと同等の数値精度を得た。

一方、ML63Q25x7向けSolist-AIライブラリの実機検証では、1インスタンス内で推論ごとに
alphaを交換しても次段の行列へ切り替わらず、各段で再初期化すると出力が0になる挙動を
確認した。このため、多段ネットワークをSolistアクセラレータだけで連続実行する構成は
現時点では採用しない。

同じBfloat16重みをMCUの逐次積和で実行した場合は、8個のgolden入力すべてでPC基準値と
一致したが、1推論あたり約0.93秒を要した。PC推論で座標回帰を実現できることは既知で
あるため、このCPU方式も製品ファームには組み込まず、基板は検証済み分類ファームへ戻した。

## 評価条件

- 4セッション、1,534波形
- leave-one-session-out 4-fold
- 25.6 kHz・512点
- 時間波形64値＋FFT帯域64値
- 全foldで学習セッションだけから標準化係数を算出
- 入力、重み、各中間出力、最終出力をBfloat16化
- Seed 1の単独モデル
- 教師座標は8エリア中心点のみ（任意位置の補間精度は未検証）

再現用コードと生成ヘッダは保管するが、`S_AcrylicPan.subdir.mk` からXY多段推論は外して
あり、通常のファームウェアビルドには含まれない。
