# PC側・確率分布型位置推定

## 構成

「位置推定」タブは、ファームが返す25.6 kHz・512点・20 msの`INFERENCE_EVENT`を使う。
PC側では時間64＋FFT 64特徴を作り、128－256－128－64－2の直接XY MLPを3初期値で推論する。

- 推定座標: 3モデルのXY平均
- モデル不確実性: 3モデルのXY標準偏差
- 空間的な第二候補: ファームの8エリアスコアを温度付きsoftmaxで確率化
- ヒートマップ: XY周辺Gaussian 72%＋8中心の確率混合Gaussian 28%
- 分布幅: 中心点LOSO誤差、クラス確率エントロピー、モデル間ばらつきから算出

モデルが存在しない場合や波形契約が一致しない場合は、8エリア確率だけを使った表示へ
フォールバックする。

## 中心点データで行った改善

現行の80 ms学習波形の先頭512点を使い、ライブファームと同じ20 ms入力契約でモデルを
再学習した。4セッション・1,534件をセッション単位LOSOで評価したSeed 1モデルは次の結果だった。

| 指標 | 結果 |
|---|---:|
| 平均距離 | 9.39 mm |
| 中央値 | 6.06 mm |
| 90%点 | 19.15 mm |
| 25 mm以内 | 94.00% |
| 50 mm以内 | 98.24% |
| エリア正解率 | 98.50% |

これは別セッションにおける8中心座標の復号精度である。中心間の任意位置を評価した数値ではない。
任意位置の出力は、全層学習NNの連続写像と空間確率分布による補間として提供する。
