# 4セッション・サンプリング周波数比較とXYモデル評価

## 結論

2026-07-18に選別を完了した中心打点4セッション、合計1,534件を使用した。
セッション単位のleave-one-session-out（LOSO）4-foldで評価した結果、8クラス分類は
**6.4 kHz・FFT 128入力**が平均正解率 **98.63%** で最良だった。

XY直接回帰は **6.4 kHz・時間64＋FFT64入力**が最良だったが、平均距離誤差は
**47.15 mm**、XYから求めたエリア正解率は **69.95%** だった。現在の教師座標は
8エリア中心だけなので、これは8中心点への粗い回帰評価であり、未学習座標への補間性能ではない。
同じデータでは、8クラス判定後にエリア中心座標へ変換する方式が平均距離
**2.41 mm**相当となり、現段階では明確に優れている。

## サンプリング変換

原記録は25.6 kHz・2,048点・80 ms・trigger index 64。12.8 kHzと6.4 kHzは
`scipy.signal.resample_poly`、Kaiser beta 8.0、`padtype=line`でアンチエイリアス処理後に
1/2・1/4分周した。単純間引きは使用していない。

この比較は同じADC原記録のオフライン変換であり、KX134自体を低いODRで動かした場合の
アナログ帯域、ノイズ、SNR変化までは再現しない。実機実装では25.6 kHz取得を維持し、
MCU側FIR＋1/4分周で6.4 kHz特徴を作る方針が安全である。

## 次の判断

現時点の実機候補は6.4 kHz・FFTの8クラスモデル。XY直接回帰は性能不足のため実機候補にしない。
XYを再評価するには、Bパターンの四隅や50 mm格子など、8中心以外の座標教師を学習に含め、
別セッションの未学習座標を評価用に確保する必要がある。

## PC側NNによるモデル容量の再評価

同じ4セッション・同じLOSO分割・同じ6.4 kHz時間＋FFT特徴を使い、Solist AIの固定射影制限を外した
全層学習MLPを追加評価した。128－256－128－64－2の直接XYモデルを3初期値でアンサンブルすると、
平均距離は47.15 mmから5.68 mmへ87.9%低下した。詳細は
[PC側・中心点XY座標回帰モデルの比較](pc-xy-regression.md)を参照する。

この結果は8中心座標の復号についてモデル表現力が主要な制約だったことを示す。ただし教師座標は
依然として8中心だけであり、未学習の中間座標への補間性能は示さない。
