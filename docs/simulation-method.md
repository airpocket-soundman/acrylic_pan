# Acrylic Pan 振動シミュレーション手法

## 位置づけ

現在のモデルは有限要素法（FEM）ではなく、Kirchhoff-Love薄板曲げモデルの有限差分法
（FDM）である。400 × 200 × 2 mmのアクリル板について、非対称固定による低～中次
固有モードと、中央1センサに現れる打撃位置差を設計段階で比較する。

## 実行環境

- Docker Desktop Linux Engine 28.0.4
- `python:3.12-slim`
- Python 3.12
- NumPy 2.2.6
- SciPy 1.15.3
- Matplotlib 3.10.3
- 実装: `sim/modal_plate.py`
- コンテナ定義: `Dockerfile`, `requirements.txt`, `run-simulation.sh`

## 形状と材料

| パラメータ | 値 |
|---|---:|
| 幅 W | 0.400 m |
| 高さ H | 0.200 m |
| 厚さ h | 0.002 m |
| ヤング率 E | 3.2 GPa |
| ポアソン比 ν | 0.35 |
| 密度 ρ | 1180 kg/m³ |
| 曲げ剛性 D | 約2.43 N·m |

曲げ剛性は `D = Eh³ / {12(1−ν²)}` とする。材料は線形、等方、均質、温度一定を仮定する。

## 離散化

- 81 × 41節点、合計3321節点
- X方向、Y方向とも5 mm間隔
- 各節点は板面垂直方向たわみ `w` の1自由度
- 5点差分Laplacian `L`
- 節点面積 `ΔA = ΔxΔy`
- 剛性行列 `K = DΔA LᵀL`
- 集中質量行列 `M = ρhΔA I`

これは薄板方程式 `D∇⁴w + ρh ẅ = 0` の空間離散化に相当する。

## 境界条件

- 固定面: `x=200..300 mm`, `y=0..20 mm`
- 固定面の21 × 5 = 105節点で `w=0`
- 3216自由節点
- 固定面以外の板外周は差分演算子の隣接欠損により自由端を近似

固定面では回転自由度を個別に持たない。20 mm幅の複数節点を固定することで、実物の
挟込み面におけるたわみと傾きの抑制を近似している。

## 固有値解析

一般化固有値問題 `Kφ=λMφ` をSciPy `eigsh`で解く。

- shift-invert: `sigma=0.0`
- selection: `which="LM"`
- 低い側から18モードを採用
- `f=√λ/(2π)`
- 応答計算には質量正規化固有ベクトルを使用
- モード形状SVGでは各モードの最大絶対値を1へ正規化

## 打撃とセンサ応答

打撃は次の8中心点に与える単位点インパルスとする。

`(50,50), (150,50), (250,50), (350,50), (50,150), (150,150), (250,150), (350,150) mm`

センサ点は1個、`(200,100) mm`。モード別のセンサ加速度振幅は次の比例式を使う。

`A_m ∝ |φ_m(hit) φ_m(sensor) ω_m|`

各打撃の棒グラフは、その打撃内で最大の結合強度を1にする。

## 空間RMS振動強度

18モードを位相非相関として時間平均し、次式で空間強度を得る。

`a_RMS(x,hit) = sqrt(0.5 Σ_m [φ_m(x) φ_m(hit) ω_m]²)`

8枚すべての共通最大値を1に正規化する。図の色とpeak値は8打撃間で比較可能だが、
単位はm/s²ではなく相対値である。

## 出力

- `results.json`: 条件、固有振動数、モード結合、打撃別peak
- `simulation-data.js`: HTML連動用データ
- `mode-1.svg`～`mode-8.svg`: 固有モード形状
- `hit-c4.svg`等8枚: 打撃別空間RMS強度
- `solist-feasibility.json`: 1隠れ層モデルの合成データ予備評価

## 制限と次段階

現在は、材料減衰、空気放射、センサ基板と接着層の質量・剛性、固定具の有限剛性、
摩擦、締付圧、打撃子接触、面内振動、温度・材料ばらつきを含まない。

次段階では次を実施する。

1. 2.5 mm格子で固有振動数とモード形状の収束確認。
2. 実測の周波数応答からモード別減衰比を同定。
3. センサ基板質量と固定具剛性を追加。
4. 必要に応じてMindlin板要素または3次元ソリッドFEMへ移行。
5. 実測FRFとMAC（Modal Assurance Criterion）でモード形状を照合。
