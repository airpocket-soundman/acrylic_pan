# シミュレーション使用ソフト・実行条件

## 使用ソフトウェア

本プロジェクトの振動解析は市販FEMソフト、FEniCS、CalculiXを使用していない。Pythonで実装した専用ソルバを、プロジェクト専用Dockerコンテナ内で実行している。

| 分類 | ソフトウェア・バージョン | 用途 |
|---|---|---|
| コンテナ | Docker Desktop / Engine 28.0.4、API 1.48 | Linux解析環境の再現とCVATからの分離 |
| OS・CPU環境 | Linux/amd64、Debian Trixie系 | コンテナ実行環境 |
| ベースイメージ | `python:3.12-slim@sha256:423ed6ab…199fbf` | 解析ランタイムの固定 |
| Python | 3.12.13 | ソルバ、後処理、データ出力 |
| NumPy | 2.2.6 | 配列・行列・FFT |
| SciPy | 1.15.3 | 疎行列、ARPACK固有値ソルバ `eigsh` |
| Matplotlib | 3.10.3 | SVG静止画、モード図、応答図、動画フレーム |
| FFmpeg | 7.1.5-0+deb13u1 | H.264/yuv420p MP4動画の生成 |
| Web表示 | HTML5、CSS、Vanilla JavaScript、SVG | グラフ、打点切替、ページ表示 |

検証時の専用Dockerイメージは `acrylic-pan-solid-fem:local`、Image IDは `sha256:7c37bb12…3782394`。Image IDはソース変更による再ビルドで変化するため、再現条件としてはDockerfile、ベースイメージdigest、固定した依存バージョンを使用する。

## 2D板モデル

| 条件 | 値 |
|---|---|
| 実装 | `sim/modal_plate.py` |
| 解析種別 | Kirchhoff–Love薄板の有限差分法（FDM） |
| 形状 | 400 × 200 × 3 mm |
| 格子 | 81 × 41節点、5 mm間隔 |
| 自由度 | 面外変位1自由度/節点、固定後3,216自由節点 |
| 剛性 | 5点Laplacianを用いた双調和演算子 `K=DΔA LᵀL` |
| 質量 | 集中質量 `M=ρhΔA I` |
| 固定 | x=200–300、y=0–20 mmに含まれる105節点の面外変位を固定 |
| 固有値解析 | `scipy.sparse.linalg.eigsh`、shift-invert、σ=0、先頭18モード |
| 用途 | 固定位置・打点位置の高速な設計比較 |

2D版は有限要素法ではなく有限差分法である。薄板曲げの設計比較には軽量だが、板厚内応力、面内変位、せん断変形は直接表現しない。

## 3DソリッドFEM

| 条件 | 値 |
|---|---|
| 実装 | `sim/solid_fem.py` |
| 解析種別 | 3次元微小変形・線形弾性FEM |
| 要素 | 8節点六面体HEX8、3並進自由度/節点 |
| 数値積分 | 2 × 2 × 2 Gauss完全積分 |
| メッシュ | 32 × 16 × 2 = 1,024要素 |
| 節点・自由度 | 1,683節点、5,049自由度 |
| 要素寸法 | 12.5 × 12.5 × 1.5 mm |
| 質量 | 節点集中質量 |
| 固定 | x=200–300、y=0–20、z=-1.5–1.5 mmに含まれる54節点のXYZ変位を固定 |
| 固有値解析 | `eigsh`、shift-invert、σ=0、先頭16モード |
| 制限 | 一次完全積分要素のせん断ロッキング、完全固定、線形材料、小変形 |

## 共通材料・打撃・センサ条件

| 条件 | 値 |
|---|---|
| 材料 | 等方PMMA |
| ヤング率 | 3.2 GPa |
| ポアソン比 | 0.35 |
| 密度 | 1,180 kg/m³ |
| センサ | 中央1個、 `(200, 100) mm`、Z方向 |
| 打点 | x=50/150/250/350、y=50/150 mmの8点 |
| 加振 | 各点へのZ方向単位力積 |
| モード減衰比 | 1.2% |
| 過渡応答 | 質量正規化モードの重ね合わせ |

センサ基板、接着層、空気放射、温度依存性、材料ばらつき、打撃接触時間、クランプ摩擦・締付け圧は現モデルに含めない。

## 中央センサ波形・FFT条件

| 条件 | 値 |
|---|---|
| サンプリング | 6,400 Hz |
| 記録長 | 2,048点、320 ms |
| FFT | 2,048点、片側振幅スペクトル |
| 窓 | Hann |
| 周波数分解能 | 3.125 Hz |
| ページ表示範囲 | 0–1,200 Hz |
| 正規化 | 時間波形・FFTそれぞれ8打点共通最大値 |

現在の波形は単位力積に対する相対値であり、絶対加速度m/s²ではない。実測との絶対比較には打撃力とKX134感度の校正が必要。

## Docker分離条件

| 項目 | 条件 |
|---|---|
| イメージ | `acrylic-pan-solid-fem:local` |
| コンテナ | `acrylic-pan-solid-fem-run` |
| ネットワーク | `--network none` |
| ルートFS | `--read-only` |
| 一時領域 | `/tmp`をtmpfsとして256 MB |
| 書込み先 | `web/assets/simulation`だけをbind mount |
| 終了処理 | `--rm` |

この構成はCVATのcomposeプロジェクト、コンテナ、イメージ、ネットワーク、ボリュームを参照・変更しない。

## 再現方法

```powershell
.\run-solid-fem.ps1
```

スクリプトはDockerfileから専用イメージをビルドし、2D板モデル、3DソリッドFEM、時間応答、FFT、SVG、JSON、MP4を生成する。詳細な式と制約は `docs/simulation-method.md`、`docs/solid-fem.md`、`docs/sensor-response.md`を参照する。
