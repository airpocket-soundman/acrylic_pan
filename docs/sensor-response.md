# 中央センサ時間波形・FFT仕様

## 観測条件

中央 `(200, 100) mm` に取り付ける1個の加速度センサについて、Z方向の単位力積応答をモード重ね合わせで計算する。複数センサは使用しない。

| 項目 | 設定 |
|---|---:|
| サンプリング周波数 | 6,400 Hz |
| サンプル数 | 2,048点 |
| 記録時間 | 320 ms |
| FFT点数 | 2,048 |
| 周波数分解能 | 3.125 Hz |
| 窓関数 | Hann |
| 表示周波数 | 0–1,200 Hz |
| モード減衰比 | 1.2% |

2D板モデルは18モード、3DソリッドFEMは16モードを使用する。打点は `(50/150/250/350, 50/150) mm` の8点で、各打点へZ方向単位力積を与える。

## 計算

質量正規化された固有ベクトルについて、打点とセンサの符号付き参加係数を次で求める。

```text
c_m = φ_m(hit) φ_m(sensor)
```

減衰自由振動のモード座標を2階微分し、中央センサの加速度へ重ね合わせる。FFTは時間波形へHann窓を掛けた後の片側振幅スペクトルとする。

時間波形は全8打点の共通最大加速度、FFTは全8打点の共通最大スペクトルでそれぞれ正規化する。このため打点間の相対強度は保持されるが、表示値はm/s²の絶対値ではない。

## 出力ファイル

- `web/assets/simulation/sensor-response-2d.json/js`: Kirchhoff板モデル
- `web/assets/simulation/solid3d/sensor-response-3d.json/js`: 3D HEX8ソリッドFEM
- `web/js/sensor-charts.js`: 打点切替、時間波形、FFTの共通表示処理

## 実測との対応

実機の採取ファームも6.4 kHz・2,048点のバーストを使用するため、同じ時間軸・FFTビンで比較できる。実測との比較では、打撃力、KX134の感度、センサ取付方向を校正した後に絶対加速度へ変換する。また、モデルに含まれないセンサ基板・接着層・クランプ摩擦によってピーク周波数と減衰が変化する点に注意する。
