# CalculiX高周波・50 ms窓解析

## 目的

KX134-1211を最大ODR 25.6 kHzで使用し、HPF後の短周期振動から打点を推論する50 ms窓を評価する。従来の30モード解析は540.2 Hzまでのため、高周波系は別ディレクトリと入力デッキで保存する。

## 解析条件

| 項目 | 値 |
|---|---:|
| ソルバー | CalculiX CrunchiX 2.20 / SPOOLES |
| 要素 | C3D20R、3次元20節点二次ソリッド、縮約積分 |
| パネル | 400 × 200 × 2 mm PMMA |
| 材料 | E=3,200 MPa、ν=0.35、ρ=1.18×10⁻⁹ tonne/mm³ |
| 固定 | x=200–300、y=0–20、z=−1–1 mmの全自由度 |
| メッシュ | 面内10 mm、板厚1 mm、1,600要素、9,285節点 |
| 固有値要求 | 280モード、0–6,000 Hz |
| 実取得 | 274モード、最高5,992.405 Hz |
| 過渡窓 | 25,600 Hz、39.0625 µs/点、50 ms、1,280点 |
| モード減衰 | 全モード 1.2% |
| 打撃 | 8打点のZ方向単位力積 |
| 観測 | 中央1センサ、(200,100,1) mmのZ加速度 |

50 ms窓の真のFFT分解能は20 Hzである。1,024点へのゼロパディングは表示ビンを細かくするが、物理的な分解能は向上させない。

## HPF予備評価

8打点の正規化FFTプロファイル間のコサイン距離を比較した。5 kHz LPFを共通とする。

| HPF [Hz] | 最悪ペア距離 | 平均ペア距離 |
|---:|---:|---:|
| 0 | 0.1858 | 0.2657 |
| 100 | 0.1862 | 0.2655 |
| 250 | 0.1783 | 0.2636 |
| 500 | 0.1767 | 0.2578 |
| 750 | 0.1839 | 0.2519 |
| 1,000 | **0.1886** | **0.2673** |

この理想FEM結果では1 kHz HPFが僅かに最良だが、差は小さい。実測ではセンサノイズ、取付共振、打撃材、打撃強度のばらつきが加わるため、100–1,000 Hzを候補として収録セッション単位の交差検証で選ぶ。

## メッシュの限界

PMMA薄板の5 kHz曲げ波長は概算約36 mmである。10 mm二次要素は予備評価には使えるが、高周波固有値の確定値とはしない。次段階で面内5 mmメッシュを使い、固有振動数、中央センサFFT、HPF判別スコアの収束を確認する。モード順のみでは交差を誤判定するため、固有ベクトルを出力してMACで対応付ける。

## 再現

`calculix/generate_highfreq_model.py`、`calculix/postprocess_highfreq.py`、`calculix/run-highfreq.sh`を専用イメージ`acrylic-pan-calculix:2.20`内で使う。実行コンテナはネットワーク無効、読み取り専用root、出力先`web/assets/simulation/calculix-highfreq`のみ書き込み可とする。

```powershell
.\run-calculix-highfreq.ps1
```
