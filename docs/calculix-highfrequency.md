# CalculiX高周波・50 ms窓解析

## 目的

KX134-1211を最大ODR 25.6 kHzで使用し、HPF後の短周期振動から打点を推論する50 ms窓を評価する。3 mm標準解析の30モードは804.9 Hzまでのため、高周波系は別ディレクトリと入力デッキで保存する。

## 解析条件

| 項目 | 値 |
|---|---:|
| ソルバー | CalculiX CrunchiX 2.20 / SPOOLES |
| 要素 | C3D20R、3次元20節点二次ソリッド、縮約積分 |
| パネル | 400 × 200 × 3 mm PMMA |
| 材料 | E=3,200 MPa、ν=0.35、ρ=1.18×10⁻⁹ tonne/mm³ |
| 固定 | x=200–300、y=0–20、z=−1.5–1.5 mmの全自由度 |
| メッシュ | 面内10 mm、板厚1.5 mm × 2要素、1,600要素、9,285節点 |
| 固有値要求 | 280モード、0–6,000 Hz |
| 実取得 | 198モード、最高5,967.227 Hz |
| 過渡窓 | 25,600 Hz、39.0625 µs/点、50 ms、1,280点 |
| モード減衰 | 全モード 1.2% |
| 打撃 | 8打点のZ方向単位力積 |
| 観測 | 中央1センサ、(200,100,1.5) mmのZ加速度 |

50 ms窓の真のFFT分解能は20 Hzである。1,024点へのゼロパディングは表示ビンを細かくするが、物理的な分解能は向上させない。

センサ加速度は変位出力の2階中心差分で求める。中心差分は1回につき振幅を`sinc(2f/fs)`倍へ減衰させる（2回で5 kHz時約0.59倍）ため、FFT振幅は`sinc²`を除算して補正している。

## HPF予備評価

8打点の正規化FFTプロファイル間のコサイン距離を比較した。5 kHz LPFを共通とする。フィルタは4次Butterworthバンドパスのゼロ位相適用（`sosfiltfilt`、実効8次）であり、実機の因果フィルタとは位相・過渡特性が異なる。HPF 0の行は無フィルタではなく内部で20 Hz–5 kHzバンドパスを適用し、距離計算の解析帯域下限は全行`max(HPF, 100) Hz`とするため、0行と100行はほぼ一致する。

| HPF [Hz] | 最悪ペア距離 | 平均ペア距離 |
|---:|---:|---:|
| 0 | 0.1960 | 0.2847 |
| 100 | 0.1952 | 0.2841 |
| 250 | 0.1933 | 0.2846 |
| 500 | **0.2004** | **0.2864** |
| 750 | 0.1868 | 0.2717 |
| 1,000 | 0.1552 | 0.2557 |

3 mmの理想FEM結果では500 Hz HPFが最悪ペア距離と平均距離の両方で最大となった。実測ではセンサノイズ、取付共振、打撃材、打撃強度のばらつきが加わるため、100–1,000 Hzを候補として収録セッション単位の交差検証で選ぶ。

## E4基準・18帯域特徴圧縮

Solist-AIの入力数を抑える候補として、HPFカットオフから5 kHzまでを18個の等幅帯域へ分けた。各帯域でFFT振幅のRMSを求め、log10化した後、E4の18特徴を差し引く。

`feature(hit, band) = log10(RMS(hit, band)) - log10(RMS(E4, band))`

振動レベルを保持する方式と、各打点の18帯域RMS振幅の和を1へ正規化してスペクトル形状だけを使う方式を比較した。距離は18特徴におけるRMSユークリッド距離であり、元の全FFTコサイン距離とは尺度が異なる。

| HPF [Hz] | レベル＋形状 最悪/平均 | 最近傍 | 形状のみ 最悪/平均 | 最近傍 |
|---:|---:|---|---:|---|
| 0（内部20 Hz） | 0.1354 / 0.2215 | G4–E5 | 0.1351 / 0.2173 | G4–E5 |
| 100 | 0.1356 / 0.2199 | G4–E5 | 0.1352 / 0.2158 | G4–E5 |
| 250 | 0.1517 / 0.2013 | G4–E5 | 0.1418 / 0.2009 | C5–D5 |
| 500 | **0.1677 / 0.2551** | G4–E5 | **0.1690 / 0.2546** | G4–E5 |
| 750 | 0.1553 / 0.2097 | D4–C5 | 0.1569 / 0.2073 | D4–C5 |
| 1,000 | 0.1651 / 0.2533 | A4–C5 | 0.1653 / 0.2506 | A4–C5 |

500 Hz HPFが両方式で最大の最悪ペア距離を得た。特徴を18個へ圧縮しても3 mm理想FEM上の打点差は残る。

ただしE4ベクトルの減算は特徴空間の平行移動であるため、ユークリッド距離そのものを強化しないことも数値確認した。E4基準化はゼロ点の明確化やセッション共通成分の補正には利用できるが、識別余裕は実測の同一クラス内距離に対するクラス間距離で評価する必要がある。

## メッシュの限界

3 mm PMMA薄板の5 kHz曲げ波長は概算約44 mmである。10 mm二次要素は予備評価には使えるが、高周波固有値の確定値とはしない。次段階で面内5 mmメッシュを使い、固有振動数、中央センサFFT、HPF判別スコアの収束を確認する。モード順のみでは交差を誤判定するため、固有ベクトルを出力してMACで対応付ける。

## 再現

`calculix/generate_highfreq_model.py`、`calculix/postprocess_highfreq.py`、`calculix/run-highfreq.sh`を専用イメージ`acrylic-pan-calculix:2.20`内で使う。実行コンテナはネットワーク無効、読み取り専用root、出力先`web/assets/simulation/calculix-highfreq`のみ書き込み可とする。

```powershell
.\run-calculix-highfreq.ps1 -ThicknessMm 3
```

5 mm候補との比較は次を実行し、`web/assets/simulation/calculix-thickness-comparison`へ比較JSONとSVGを生成する。

```powershell
.\run-calculix-thickness-comparison.ps1
```
