# 打撃トリガー閾値の再評価（2026-07-18）

クリーニング済み4セッション、1,534打撃を対象に、25.6 kHzの原波形でトリガー直後と残響区間を再評価した。

## 対象セッション

- `20260718_074916_84b183d2`
- `20260718_080557_9524983c`
- `20260718_081909_9dd2d785`
- `20260718_084143_ac8e6d56`

## 観察結果

- 現行の一次条件は隣接差分1,000 LSB、絶対値200 LSBである。
- 32 g設定では静止Z軸重力が約1,024 LSBになるため、絶対値200 LSB条件は常時成立し、実質的に差分条件だけで発火する。
- 静止強制測定8件の最大隣接差分は1,024 LSB、基準からの最大振幅は1,177 LSBだった。
- 実打撃のトリガー後16サンプル以内の基準相対ピークは、1%点2,894 LSB、5%点4,191 LSB、中央値11,568 LSBだった。
- 基準相対2,000 LSBを確認条件にすると、実打撃1,534件中99.54%を維持し、保存済み静止測定は全件除外できる。

## 採用方式

モデル入力の時間位置を変えないため、一次差分閾値は1,000 LSBのままとする。一次条件を満たした後、16サンプル（0.625 ms）以内に直前64サンプル平均から2,000 LSB以上離れた場合だけ打撃を確定する。未達の候補はイベント化せず破棄する。

残響自体は大振幅になるため、この振幅確認だけでは完全には除外できない。楽器画面の「連打の最小間隔」と併用する。
